GeoExt.ux.tree.LayerTreeBuilder UX

This tree automatically builds nodes based on the layer records in a GeoExt.data.LayerStore. The layer 'group' property determines where to put the layer node and creates the missing nodes when needed.

In this example, the "Parks" layer has the 'group' property set to "GMap/Polygons". The "/" character is the subfolder delimiter. There are various other interesting properties a layer can have, such as hideInLegend, and the tree itself too.

The javascript file used in this example is not minified so it is readable. See tree-builder.js.
